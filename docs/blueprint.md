# **App Name**: MessMate

## Core Features:

- Daily Menu Display: Display today's menu with breakfast, lunch, and dinner options.
- Meal Feedback: Allow students to submit meal feedback with ratings and comments. Limit one submission per meal per student.
- Meal Ordering: Enable students to place orders for meals within a specified timeframe.
- Admin Interface: Admin tool for updating the daily menu, view orders, and review feedback.
- Feedback Analysis: Use AI to summarize and categorize feedback to help the manager determine trends. AI tool should detect negative comments, and consider positive feedback when making suggestions for improvement

## Style Guidelines:

- Primary color: Warm and inviting orange (#FF9800) to evoke feelings of comfort and appetite.
- Background color: Light, desaturated orange (#FFF3E0) for a clean and unobtrusive backdrop.
- Accent color: Yellow (#FFEB3B) to highlight key actions and information.
- Body and headline font: 'PT Sans', a modern sans-serif for readability and a friendly feel.
- Use food-related icons for menu items and interface elements.
- Clean and intuitive layout with clear sections for menu, ordering, and feedback.
